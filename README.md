# 健身增肌减脂打卡

一个纯前端的健身数据记录页面，支持体重、体脂、喝水、运动和饮食打卡。

## 使用方法

1. 用浏览器直接打开 `index.html`。
2. 数据会自动保存在当前浏览器的 `localStorage` 中，无需登录。
3. 底部导航可切换：趋势 / 今日打卡 / 日历历史。
4. 在日历中点击任意日期，可补填或修改那一天的记录。

## 文件说明

- `index.html`：页面结构
- `css/style.css`：自定义样式
- `js/app.js`：业务逻辑与数据持久化

## 注意事项

- 数据仅保存在本机浏览器，清除浏览器数据或换设备会丢失。
- 如需多设备同步，后续可扩展后端服务。
